# Plan 01: Feature Integration

<objective>
Wire the prompt library insertion mechanism to all feature components using a custom `prompt-inserted` event.
</objective>

<task>
<read_first>
- src/components/modals/PromptLibraryModal.tsx
</read_first>
<action>
In `src/components/modals/PromptLibraryModal.tsx`:
Modify the "Use Prompt" button click handler to dispatch a custom event on the `window` object before closing the modal.
```typescript
onClick={(e) => {
  e.stopPropagation();
  window.dispatchEvent(new CustomEvent('prompt-inserted', { detail: { prompt: prompt.text } }));
  if (onSelect) onSelect(prompt.text);
  showToast(t('promptLibrary.usePrompt') || 'Prompt applied');
  onClose();
}}
```
</action>

<task>
<read_first>
- src/components/VirtualTryOn.tsx
</read_first>
<action>
In `src/components/VirtualTryOn.tsx`, inside the component (before the return statement), add a `React.useEffect` hook to listen to the `prompt-inserted` event and update `setExtraPrompt(e.detail.prompt)`. Add `setExtraPrompt` to the dependency array of the hook.
```typescript
  React.useEffect(() => {
    const handlePromptInserted = (e: Event) => {
      const customEvent = e as CustomEvent<{ prompt: string }>;
      setExtraPrompt(customEvent.detail.prompt);
    };
    window.addEventListener('prompt-inserted', handlePromptInserted);
    return () => window.removeEventListener('prompt-inserted', handlePromptInserted);
  }, [setExtraPrompt]);
```
</action>
