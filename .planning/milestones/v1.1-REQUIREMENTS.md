## v1.1 Requirements

### Virtual Try-On

- [x] **VTO-BATCH-01**: User can batch multiple subject images in Virtual Try-On with one shared outfit set
- [x] **VTO-BATCH-02**: One request is created per subject image and outfit inputs are reused rather than expanded into cross-products

### Clothing Transfer

- [x] **CTR-BATCH-01**: User can batch multiple concept images in Clothing Transfer with one shared reference outfit set
- [x] **CTR-BATCH-02**: Every Clothing Transfer request preserves `images: [concept, ...references]`

- [x] **UX-BATCH-01**: User can see per-item processing status, partial failures, and completed outputs without losing the single-image path
