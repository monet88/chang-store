# Plan 12.02: Multi-Person UI — Toggle, Overlay, Clear

<objective>
Add the multi-person mode toggle pill, marker canvas overlay, and clear button to `VirtualTryOn.tsx`. Add i18n keys to both locale files. No business logic — all state comes from the hook extended in Plan 12.01.
</objective>

<must_haves>
- Amber pill toggle placed inside Step 1 panel, below subject image uploader.
- Clicking subject image while mode is active places a red dot marker (24×24px, white border, centered on click point).
- "Clear marker" button appears only when `markerPosition !== null` — same ghost style as "Clear subjects".
- Toggling mode OFF automatically clears the marker (handled in hook, no extra UI logic needed).
- All user-facing strings use `t()` with keys added to `en.ts` / `vi.ts`.
- No business logic in the component — only `useState` for local UI state if needed (none expected).
</must_haves>

<task>
<description>Add marker overlay and click handler to the selected subject image area</description>
<read_first>
- src/components/VirtualTryOn.tsx
- src/components/MultiImageUploader.tsx
</read_first>
<action>
In `src/components/VirtualTryOn.tsx`, wrap the `<MultiImageUploader>` in a `<div className="relative">` and overlay a click-capture layer when multi-person mode is active.

Replace the existing `<div className="space-y-2">` wrapping the MultiImageUploader:

```tsx
<div className="space-y-2">
  {/* Zona de imagen con overlay para marcador multi-persona */}
  <div className="relative">
    <MultiImageUploader
      images={subjectImages}
      id="subject-upload"
      title={t('virtualTryOn.subjectImagesTitle')}
      onImagesUpload={handleSubjectImagesUpload}
    />
    {/* Overlay de captura de click — solo activo en modo multi-persona con imagen cargada */}
    {isMultiPersonMode && subjectImages.length > 0 && (
      <div
        id="multi-person-overlay"
        className="absolute inset-0 cursor-crosshair z-10"
        onClick={(e) => {
          const rect = e.currentTarget.getBoundingClientRect();
          const x = e.clientX - rect.left;
          const y = e.clientY - rect.top;
          // Normalizar a coordenadas relativas 0-1 para uso en Phase 13
          const relX = x / rect.width;
          const relY = y / rect.height;
          setMarkerPosition({ x, y, relX, relY });
        }}
      />
    )}
    {/* Punto de marcador rojo con borde blanco */}
    {isMultiPersonMode && markerPosition && (
      <div
        id="multi-person-marker"
        className="absolute z-20 pointer-events-none"
        style={{
          left: markerPosition.x,
          top: markerPosition.y,
          width: 24,
          height: 24,
          borderRadius: '50%',
          backgroundColor: '#ef4444',
          border: '2px solid white',
          transform: 'translate(-50%, -50%)',
          boxShadow: '0 0 0 2px rgba(239,68,68,0.4)',
        }}
        aria-label="Multi-person target marker"
      />
    )}
  </div>
  {subjectImages.length > 0 && (
    <button
      onClick={clearSubjectImages}
      disabled={isLoading}
      className="w-full text-xs text-zinc-400 hover:text-red-400 border border-zinc-700 hover:border-red-500/50 rounded-lg py-1.5 transition-colors disabled:opacity-50 disabled:cursor-not-allowed"
    >
      {t('virtualTryOn.clearSubjects')}
    </button>
  )}
</div>
```
</action>
